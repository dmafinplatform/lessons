Это Static - обычный элемент
scroll - остается на месте при прокрутке
Relative - устанавливается в контексте со смещением
Absolute - позиционируется относительно ближайшего родителя с position: relative

`float` для изображения и текст будет обтекать изображение, создавая красивый макет.

Fixed - фиксируется на сайте без движения
