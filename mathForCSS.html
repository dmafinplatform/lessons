Таблица умножения

Выбираем максимальные значения множителей

Например, 5*5
Либо, 6*6

Определяем порядок подсчета баллов

Например, за каждое правильное решение по 2 балла
Либо, за каждое правильное решение по 3 балла

Далее идет игра

По кругу игрок выставляет ответ, задавая вопрос "Сколько будет X * Y?"

Окончание игры

Игра заканчивается, когда один из игроков выставляет свой последний ответ на фишке

Подсчет баллов

Каждый игрок подсчитывает свои баллы в конце игры, основываясь на правильных ответах.
